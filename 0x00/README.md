## 환경 세팅

> ```cpp
> #include <bits/stdc++.h>
> ```

- `bits` 폴더 안의 `stdc++.h`에 수많은 헤더들을 미리 include 해두어 간편함

1. 터미널에서 `/usr/local/include` 경로로 이동
2. `bits` 폴더 생성
3. [여기](https://gcc.gnu.org/onlinedocs/gcc-4.8.0/libstdc++/api/a01541_source.html)에서 코드 복사
4. 경로에 `stdc++.h` 파일 만든 후 붙여넣고 저장

> [!NOTE]
>
> ```cpp
> #include <cstdalign>
> ```
>
> 오류 발생해 주석 처리함
